# Home Cinema

A application to manages video files (movies, series, anime, etc) locally to distribute in the local network.

## First Steps

### Installing Packages

```sh
pipenv install <package name>
```

### Running

```sh
# if outside venv
pipenv run python main.py

# if inside nenv
python main.py
```

### Running Flask

```sh
python -m flask --app main run
```
